# Reception Triage Guide

A web app for GP reception staff to navigate clinical triage safely.

## Quick Deploy to Vercel (Easiest)

1. **Go to GitHub**: https://github.com/new
2. **Create new repository** called `Reception-app`
3. **Upload all files** from this folder (drag and drop, or use GitHub Desktop)
4. **Go to Vercel**: https://vercel.com
5. **Sign in with GitHub**
6. **Import** your `Reception-app` repository
7. **Click Deploy** - Vercel handles everything automatically
8. **Done!** You'll get a URL like `reception-app.vercel.app`

## Features

- Step-by-step triage workflow
- RED/AMBER flag recognition
- Pharmacy First signposting
- Quick-copy scripts for patients
- Training scenarios with quiz
- Admin console for customisation
- Works on phone, tablet, desktop

## Changing Content

1. Login as admin
2. Go to Admin tab
3. Edit contacts, scripts, settings
4. Changes save automatically

## Need Help?

The app is a standalone website - staff just visit the URL in any browser. No installation needed.
